Game Operations

  • A new game is started when you launch MahJong. Each game has a unique number which can be used to replay your favorite challenge.

  • The "Game" menu bar selection allows one to start a new game (Play), restart the current game (Restart), choose to play a specific game based on its unique game number (Numbered), and exit the game (Exit).
  • The "Sound" menu bar selection allows a player to turn off or on the sound in the game.
  • The "Move" menu bar selection allows a player to undo the last move made during that game.
  • The "Help" menu bar selection allows a player view help and game operation information.

  • A player can only select tiles that are eligible to be matched. When selected, they display a red border.
  • A tile can be deselected by clicking it again to remove the red border.
